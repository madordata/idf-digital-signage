# מדריך התקנה, הגדרה והפעלה - מערכת מסכים דיגיטליים (Digital Signage)

מדריך זה מציג את השלבים המלאים להתקנה, הגדרה והפעלה של מערכת מסכי התצוגה הדיגיטליים של היחידה. המערכת מותאמת ריצה רציפה על גבי מסכי טלוויזיה (TV) בצורה עצמאית לחלוטין וללא צורך במגע ידי אדם.

---

## 📋 דרישות קדם לחומרה ותוכנה

לפני שמתחילים, יש לוודא כי המחשב עליו מותקנת המערכת (מחשב המסך או שרת מקומי) כולל:
1. **Node.js** - גרסה 18 ומעלה (מומלץ גרסה 20 LTS). ניתן להוריד מהאתר הרשמי: <https://nodejs.org>.
2. **דפדפן אינטרנט** - Google Chrome (מומלץ במיוחד) או Microsoft Edge מעודכן.
3. **חיבור לאינטרנט / רשת מקומית** - לצורך התקנה ראשונית ועדכון נתונים מ-Google Sheets (במידה ומשתמשים בו).

---

## 🛠️ שלב 1: התקנת המערכת

1. **העתקת תיקיית הפרויקט:** העתיקו את תיקיית המערכת המלאה למיקום קבוע במחשב המיועד (לדוגמה: `C:\idf-digital-signage`).
2. **פתיחת תיקיית המקור:** ודאו שכל קבצי המפתח נמצאים שם, כולל קובץ הקיצור של ההפעלה [launch-signage.bat](launch-signage.bat).

---

## ⚙️ שלב 2: הגדרת קובץ קונפיגורציה (.env)

המערכת משתמשת בקובץ הגדרות סביבו כדי לעבוד. 
1. בתיקייה הראשית של הפרויקט, מצאו את הקובץ [.env.example](.env.example).
2. שנו את שם הקובץ ל-[.env](.env) (או העתיקו אותו וצרו עותק חדש בשם זה).
3. פתחו את הקובץ [.env](.env) בעזרת פנקס רשימות (Notepad) והגדירו את המשתנים הבאים:

   - `VITE_API_URL` - כתובת ה-API של השרת האחורי. כברירת מחדל הכל רץ מקומית: `http://localhost:4100`.
   - `PORT` - הפורט עליו ירוץ שרת הנתונים (ברירת מחדל: `4100`).
   - `GOOGLE_SHEET_ID` - מזהה ייחודי של גיליון ה-Google Sheets שברצונכם לחבר (אופציונלי). השאירו ריק אם ברצונכם להשתמש בקובץ ה-Excel המקומי [server/updates.xlsx](server/updates.xlsx).

---

## 📊 שלב 3: ניהול תוכן המסכים (מקומי או ענן)

ישנן שתי דרכים לעדכן את התכנים המוצגים במסכים (המלל של הטיקר הרץ ושל ששת המסכים):

### אפשרות א': שימוש בקובץ Excel מקומי (מומלץ להתקנות ללא אינטרנט)
1. השרת קורא הודעות ועדכונים מקובץ Excel בנתיב [server/updates.xlsx](server/updates.xlsx).
2. אם הקובץ לא קיים, המערכת תייצר אותו אוטומטית עם תכני ברירת מחדל בעת ההפעלה הראשונה.
3. ניתן לפתוח את הקובץ בכל עת, לערוך את עמודי הטיקר והמסכים, לשמור, והתצוגה תתעדכן מעצמה.

### אפשרות ב': חיבור ל-Google Sheets (מומלץ לעדכון מרחוק ונוח)
תוכלו לנהל את כל ששת המסכים ואת הטיקר הארצי מתוך דפדפן האינטרנט מכל מחשב אחר:
1. הריצו בתיקיית הפרויקט את הפקודה המייצרת תבנית מלאה: `npm run sheet:template`.
2. קובץ התבנית יווצר בנתיב [google-sheets-template/signage-content-template.xlsx](google-sheets-template/signage-content-template.xlsx).
3. העלו את הקובץ ל-Google Drive שלכם, פתחו אותו כ-Google Sheets.
4. לחצו על כפתור **Share (שיתוף)** בפינה הימנית העליונה של הגיליון, ושנו את הגדרות הגישה ל-**Anyone with the link (כל מי שמחזיק בקישור יוכל לצפות)**.
5. העתיקו את הקישור מכתובת הדפדפן, חלצו מתוכו את ה-ID הייחודי (הקוד הארוך שמופיע בין `/d/` לבין `/edit`) והדביקו אותו בתוך הקובץ [.env](.env) תחת `GOOGLE_SHEET_ID`.
6. לפרטים ומבנה העמודות המדויק, עיינו במדריך הייעודי [GOOGLE_SHEETS_SETUP.md](GOOGLE_SHEETS_SETUP.md).

## 🖥️ שלב 4: הפעלה מהירה מהשולחן העבודה (Desktop Launcher)

כדי לאפשר הפעלה קלה במחשבי היחידה שאינם מחשבי פיתוח, יצרנו עבורכם סקריפטים ייעודיים שמפעילים את השרת, פותחים את הדפדפן ומכניסים את האתר אוטומטית למצב **מסך מלא קבוע (Kiosk Mode)**:

### הפעלה במערכות הפעלה Windows:
1. מצאו את הקובץ [launch-signage.bat](launch-signage.bat) בתיקייה הראשית של המערכת.
2. לחצו עליו קליק ימני ובחרו ב-**Send to (שלח אל) -> Desktop (shortcut) / שולחן עבודה (צור קיצור דרך)**.
3. קיצור דרך ייעודי של המכשיר יופיע על שולחן העבודה שלכם.
4. **לחיצה כפולה על קיצור הדרך** תשלים את כל הפעולות הנדרשות: תפנה פורטים תפוסים, תריץ את השרת המקומי והממשק המעוצב, ותפתח את Google Chrome או Microsoft Edge במסך מלא מאובטח המותאם במיוחד לתצוגת טלוויזיה!

### הפעלה במערכות הפעלה macOS או Linux:
1. מצאו את הקובץ [launch-signage.sh](launch-signage.sh) בתיקיית הפרויקט.
2. העניקו לו הרשאות הרצה בטרמינל פעם אחת בלבד:
   ```bash
   chmod +x launch-signage.sh
   ```
3. העתיקו את הקובץ או צרו לו קיצור דרך בשולחן העבודה. לחיצה כפולה תפעיל את המערכת ותפתח את הדפדפן שלכם ב-Kiosk Mode.

## 📺 שלב 5: אופטימיזציה לטלוויזיה במסכי יחידה (TV Best Practices)

כדי להבטיח נראות מושלמת ללא תקלות, בצעו את ההגדרות הבאות במחשב המחובר לטלוויזיה:

1. **הגדרות חשמל ושינה (Power Settings):**
   - הגדירו את המחשב כך ש**לעולם לא ייכנס למצב שינה** (Set "Sleep" to Never).
   - הגדירו את המסך כך ש**לעולם לא יכבה או יוחשך** (Turn off Display to Never).
2. **הסתרת סמן העכבר (Hide Mouse Cursor):**
   - היות והמערכת אינה אינטראקטיבית, מומלץ להסתיר את סמן העכבר.
   - במחשבי Windows ניתן להוריד כלי חינמי זעיר כמו **AutoHideMouseCursor** או להזיז את העכבר לפינה הקיצונית התחתונה של המסך.
3. **עלייה אוטומטית עם הפעלת המחשב (Auto-Start on Boot):**
   - כדי שהמערכת תעלה לבד לאחר הפסקת חשמל או הפעלה מחדש של המחשב:
   - ב-Windows, פתחו את תיקיית האתחול על ידי לחיצה על מקש `Windows + R`, הקלדת `shell:startup` ולחיצה על Enter.
   - העתיקו את קיצור הדרך של הקובץ [launch-signage.bat](launch-signage.bat) שיצרתם ישירות לתוך תיקיית האתחול שנפתחה.
4. **ביטול שומרי מסך ועדכוני מערכת:**
   - כבו שבירת מסכים (Screensavers).
   - הגדירו שעות פעילות של עדכוני Windows כך שלא יאתחלו את המחשב באמצע זמן הפעילות של היחידה.

## 📞 תמיכה ופתרון תקלות מהיר

- **הדפדפן נפתח אך מציג שגיאת התחברות?**
  ודאו שחלון הטרמינל השחור שנפתח לא נסגר. המערכת דורשת את חלון השרת הרץ ברקע.
- **הדפדפן נפתח אך לא במסך מלא?**
  לחצו קבוצה על מסך הדפדפן ולחצו על מקש **F11** במקלדת כדי להכניס את הדפדפן למצב מסך מלא באופן ידני.
- **הטיקר או המסכים לא מתעדכנים מ-Google Sheets?**
  ודאו כי הגיליון משותף לציבור במצב צפייה ("Anyone with the link can view") ושהעתקתם במדויק את מזהה הגיליון (ה-ID) הנכון בקובץ [.env](.env). המתינו לפחות דקה אחת לסנכרון מלא.
